# Prompt hỏi về hệ thống Enemy & Dungeon của Dungeon Mania

---

## 1. CẤU TRÚC FILE & SCRIPTS

Liệt kê tất cả các file trong thư mục:
- `Assets/_DungeonMania/EnemyAI/` - tất cả file .cs
- `Assets/_DungeonMania/GameManager/` - tất cả file .cs

Với mỗi file, cho biết:
- Tên class chính
- Chức năng chính
- Các biến public quan trọng

---

## 2. HỆ THỐNG ENEMY

### EnemyClass.cs
- Các loại enemy có sẵn (ID, tên, stats)?
- Cách stats được tính toán (HP, ATK, armor...)?
- Có cơ chế scaling theo level không?

### EnemyScript.cs
- Enemy AI hoạt động như thế nào?
- Các state của enemy (idle, chase, attack, hit, death)?
- Cách enemy tấn công player?
- Có ranged attack không?

### EnemyAttack.cs & EnemyDamage.cs
- Cách tính damage?
- Có critical hit không?
- Có elemental damage không?

---

## 3. HỆ THỐNG WAVE & DUNGEON

### GamePlayManager.cs
- Quản lý wave như thế nào?
- Các biến: waveOfArena, leftEnemiesForDoor, isLevel?
- Cách chuyển wave?
- Quản lý boss như thế nào?

### EnemyManager.cs
- Spawn enemy như thế nào?
- Có spawn points không?
- Quản lý số lượng enemy thế nào?

### Level.cs
- Các loại level (dungeon, arena, boss)?
- Cấu hình wave như thế nào?

---

## 4. CÁCH TÍCH HỢP VÀO GAME KHÁC

- Những class/stubs nào cần có trong project mới?
- Những prefab nào cần thiết?
- Cần setup gì trong Unity Editor?
- Có scene mẫu nào không?

---

## 5. CÁC CÂU HỎI CỤ THỂ

1. Làm sao để spawn 1 con skeleton ở vị trí cụ thể?
2. Làm sao để set số lượng enemy mỗi wave?
3. Làm sao để gọi boss ở wave cuối?
4. Khi player giết hết enemy trong wave, làm sao để sang wave tiếp theo?
5. Cần những GameObject nào trong scene để hệ thống hoạt động?

---

## 6. YÊU CẦU XUẤT CODE

Với mỗi phần, nếu có thể:
- Cho ví dụ code cách sử dụng
- Cho code mẫu spawn enemy
- Cho code mẫu setup wave

---

## 7. CÁC SỰ KIỆN & DELEGATES

- Có những event nào?
- Khi enemy chết thì gọi event gì?
- Khi wave kết thúc thì gọi event gì?

---

## 8. CẦN STUB/REPLACE NHỮNG CLASS NÀO?

Liệt kê những class từ UnityEngine/UnityEngine.UI mà game sử dụng nhưng không có trong Dungeon Mania gốc (ví dụ: PlayerHealth, PlayerManager, v.v...)
